    Hexo 博客框架 Anatolo 主题 部署教程

    准备工作

    1. 安装node.js
    2. 安装git
    3. 安装hexo

    部署教程

    1. 创建Blog文件夹，进入文件夹后，右键打开Git Bash

    2. Hexo初始化
      运行hexo init

    3. 导入Anatolo主题
      运行``git clone https://gitee.com/Lhcfl/hexo-theme-anatolo.git themes/Anatolo`

    4. 安装插件
      npm install --save hexo-renderer-pug
      npm install --save hexo-renderer-stylus
      npm install --save hexo-admin（用于在线后台管理）
      npm install --save hexo-deployer-git（用于部署网站）
      再执行一遍 npm install

    5. 将theme/Anatolo文件夹下的_config.example.yml原地复制一份副本，并改名为_config.yml

    6. 打开Blog文件夹下的_config.yml，找到最下方：

    7. Hexo构建静态页面
      运行hexo generate或者hexo g

    8. Hexo运行本地服务
      运行hexo server或者hexo s
      打开浏览器，输入localhost:4000并转到，即可看见博客页面了
      在Git Bash终端按下Ctrl + C可以终止服务

    9. Hexo推送远程仓库
      运行hexo deploy或者hexo d

    10. Hexo在线编辑博客
      在运行本地服务之后，进入localhost:4000/admin页面即可在线编辑博客文档
